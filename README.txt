Booking plugin for events.

== Description ==

Features:
* Custom events
-Repeat time, days, months
-Expiration and start date
-Max group size
-Languages
* Custom bank holidays 
* Custom Booking categories 
* Custom Color setting
* Custom Price categories (Adult, Child, Private party group,...)
* Custom Currency (101	options)
* Custom Color setting
* Email confirmations
* Anonymization user sensitive data on cancelation booking
* Customer will lose claim on him booking if the booking is not confirmed by email link in 1 hour.
